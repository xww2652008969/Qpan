# 注册

## 接口地址

```
/user/register
```

## 请求类型

`post`

## 请求参数
|   字段   |  必选  |   类型   |  说明  |
| :------: | :----: | :------: | :----: |
|  `name`  | `ture` | `string` | 用户名 |
| `passwd` | `ture` | `string` |  密码  |

## 返回值

|  字段  | 类型  |  值  |   说明   |
| :----: | :---: | :--: | :------: |
| `code` | `int` | 200  | 请求成功 |
